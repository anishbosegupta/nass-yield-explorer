# NASS Corn Yield Explorer

A Gradio web app for exploring USDA corn yield trends by state (2000–2025).

## Stack
- Python + pandas (data cleaning)
- Databricks (EDA and visualizations)
- Gradio (web app)
- Mistral AI (insight generation)
- USDA NASS Quick Stats data

## How to Run
pip install gradio plotly pandas mistralai
python app.py
